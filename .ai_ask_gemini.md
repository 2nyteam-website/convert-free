# ConvertFree 최종 보안/취약점 검증 요청

## 사이트 정보
- URL: https://www.convertfree.cc
- 기술: Next.js 16 + Tailwind + shadcn/ui + TypeScript
- 100% 클라이언트 사이드 처리 (서버 없음)
- Vercel 호스팅 (무료)
- 16개 파일 변환 툴 (이미지 8, PDF 4, 비디오 2, 기타 2)

## 이전 Gemini 검증 후 수정한 사항 (2026.03.12)
1. 배치 변환 UI 멈춤 → setTimeout yield 추가
2. 드래그앤드롭 코드 중복 → FileDropzone 컴포넌트 생성
3. PDF.js 워커 CDN 의존 → public/pdf.worker.min.mjs 로컬 복사
4. Markdown 파서 취약 → marked Lexer AST 파싱으로 교체
5. Object URL 메모리 누수 → 전체 툴 useEffect cleanup 추가
6. Canonical URL → metadataBase 추가
7. Breadcrumb JSON-LD → 전체 툴 추가
8. sitemap + robots → 생성 완료
